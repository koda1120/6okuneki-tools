### PDF解析の注意点

- テキストベースのPDFのみ対応（スキャン画像PDFは非対応）
- カード会社によって精度にばらつきがあります
- 読み取れなかった場合はCSVをお試しください

### テスト済みカード会社（PDF）

- 楽天カード: 比較的読み取りやすい
- 三井住友カード: 読み取り可能
- JCBカード: 読み取り可能
- アメックス: テーブル形式で読み取りやすい

### 読み取りが難しいケース

- 画像ベースのPDF
- 複雑なレイアウトのPDF
- 暗号化されたPDF
