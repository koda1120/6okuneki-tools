## テスト観点

### 正常系
- [ ] CSV: 各カード会社フォーマットで読み込めるか
- [ ] PDF: テキスト抽出・パース成功するか
- [ ] カテゴリ分類が適切か
- [ ] サブスク検出が正確か
- [ ] 合計金額が正しいか

### エッジケース
- [ ] 少数取引（3件）でも動作するか
- [ ] 高額取引（数十万円）でも正常表示か
- [ ] サブスク多数でも警告が出るか

## PDF解析の注意点

- テキストベースのPDFのみ対応（スキャン画像PDFは非対応）
- カード会社によって精度にばらつきがあります
- 読み取れなかった場合はCSVをお試しください

## テストファイル生成

```bash
# PDFテストファイルを再生成
npx tsx scripts/generate-test-pdfs.ts
npx tsx scripts/generate-more-pdfs.ts
```
