# クレカ明細チェッカー デザイン差別化プロンプト

## 背景

現在、6億ニキツール群には3つのツールがある：
1. 携帯プラン最適化
2. クレカ明細チェッカー（このツール）
3. 不動産仲介手数料チェッカー

**問題**: 3つのツールが「色違いの同じデザイン」になっており、ユーザーから見ると差別化ができていない。カード、ボタン、レイアウト構造がすべて同一で、アクセントカラーだけが異なる状態。

**目標**: 各ツールのテーマ・世界観に合わせた独自のデザイン言語を確立し、一目で「別のツール」とわかるようにする。

---

## このツールのコンセプト

### テーマ: 「家計簿・温かみ・気づき」

クレカ明細は**日々の生活費**を扱う。冷たいデータ分析ではなく、「あ、こんなに使ってたんだ」という**気づき**を与える温かみのあるツール。

**キーワード:**
- 家計簿・生活感
- 温かみ・親しみやすさ
- ビジュアル重視（グラフ・チャート）
- 発見・気づき

**ペルソナ:**
- 20〜40代
- 「なんとなくお金が貯まらない」
- 細かい家計簿は面倒
- ざっくり傾向を知りたい

## デザイン方針

### 1. レイアウト: ビジュアル・ストーリー型

```
【現状】結果を上から順に並べるだけ
【変更】ストーリー性のある流れに

- 結果画面を「章」のように区切る
- 各セクションに見出しと導入文をつける
- 例: 「あなたの1ヶ月の支出」→「カテゴリ別に見ると...」→「気になるサブスク」
- スクロールで自然に読み進められる構成
```

### 2. 色使い: 温かみのあるナチュラル系

```
【現状】テラコッタ + クリームベージュ
【変更】より温かみを強調、でも甘すぎない

- 背景: 温かみのあるオフホワイト (#FBF8F3)
- カード背景: クリーム系 (#FFFCF5)
- アクセント: 落ち着いたコーラル (#E07A5F)
- テキスト: 濃いブラウン (#3D2C29)
- サブテキスト: ミディアムブラウン (#8B7355)
- カテゴリ色: パステル寄りの多色（食費=オレンジ、日用品=緑、娯楽=紫など）
```

### 3. カードスタイル: ソフト・有機的

```
【現状】角丸12px、薄いシャドウ
【変更】より柔らかく

- 角丸: 16px〜20px（より丸みを）
- シャドウ: 温かみのある影 (0 4px 20px rgba(139, 115, 85, 0.08))
- ボーダー: なし（シャドウで区切り）
- カード内余白: ゆったり（20px〜24px）
- 背景に薄いテクスチャ or グラデーションを検討
```

### 4. グラフ・チャート: 主役として目立たせる

```
- 円グラフを大きく表示（画面幅の80%くらい）
- カテゴリ色は区別しやすく、でも調和のとれた配色
- グラフにアニメーション（ページ表示時にふわっと表示）
- 金額はグラフの中央に大きく表示
- 凡例はグラフの下にアイコン付きで表示
```

### 5. アイコン: 生活・お金系

```
- Lucide Reactを継続使用
- カテゴリアイコン:
  - Coffee（カフェ）, ShoppingBag（買い物）, Utensils（食事）
  - Home（住居）, Car（交通）, Smartphone（通信）
  - Tv（サブスク）, Heart（健康）, Sparkles（娯楽）
- 装飾アイコン:
  - PiggyBank（貯金）, Wallet（財布）, Receipt（レシート）
  - TrendingDown + 円マーク（節約）
```

### 6. 特徴的なUI要素

```
1. カテゴリカード（イラスト付き）
   - 各カテゴリを大きめのカードで表示
   - アイコン + カテゴリ名 + 金額 + 割合バー
   - タップで詳細展開

2. サブスク一覧（警告表示付き）
   - 月額合計を大きく表示
   - 各サービスにロゴ or アイコン
   - 「解約する」ボタンを目立たせる
   - 使っていなさそうなものに警告マーク

3. 気づきカード
   - 「今月のハイライト」的なセクション
   - 「カフェ代が先月より2,000円増えました」
   - イラスト or アイコン付きで親しみやすく

4. CSV取り込みUI
   - ドラッグ&ドロップエリアを目立たせる
   - ファイルアイコンのアニメーション
   - 「カード会社を選んでください」は大きなボタンで
```

### 7. マイクロインタラクション

```
- カードホバー時: ふわっと浮く
- ボタンタップ: 軽くバウンス
- 数字: カウントアップアニメーション
- セクション切り替え: フェードイン
```

## 実装時の注意

1. **共通コンポーネント（packages/shared）は変更しない**
   - このツール専用のコンポーネントをsrc/components/に作成

2. **tailwind.config.jsでカラーを上書き**
   ```js
   colors: {
     'bg-base': '#FBF8F3',
     'bg-card': '#FFFCF5',
     'text-main': '#3D2C29',
     'text-sub': '#8B7355',
     'accent': '#E07A5F',       // コーラル
     'accent-light': '#FEF3F0',
     'border': '#E8DFD3',
     'success': '#6A9A78',      // モスグリーン
     'warning': '#E09F3E',      // マスタード
   }
   ```

3. **Chart.jsのスタイルもカスタマイズ**
   - 現在のchartColorsをより温かみのある配色に

4. **既存機能は壊さない**
   - デザイン変更のみ、ロジックは触らない
