# 不動産仲介手数料チェッカー デザイン差別化プロンプト

## 背景

現在、6億ニキツール群には3つのツールがある：
1. 携帯プラン最適化
2. クレカ明細チェッカー
3. 不動産仲介手数料チェッカー（このツール）

**問題**: 3つのツールが「色違いの同じデザイン」になっており、ユーザーから見ると差別化ができていない。カード、ボタン、レイアウト構造がすべて同一で、アクセントカラーだけが異なる状態。

**目標**: 各ツールのテーマ・世界観に合わせた独自のデザイン言語を確立し、一目で「別のツール」とわかるようにする。

---

## このツールのコンセプト

### テーマ: 「プロフェッショナル・信頼・交渉サポート」

不動産は**高額取引**。ユーザーは「損したくない」「騙されたくない」という不安を持っている。**専門家が味方についてくれる安心感**を与えるデザイン。

**キーワード:**
- プロフェッショナル・専門性
- 信頼感・安心感
- 高級感（でも敷居は低く）
- 交渉・サポート

**ペルソナ:**
- 25〜45代
- 賃貸物件を探している
- 「仲介手数料って交渉できるの？」
- 不動産の知識はあまりない
- LINEで気軽に相談したい

## デザイン方針

### 1. レイアウト: 権威性 + 親しみやすさのバランス

```
【現状】他ツールと同じカード縦並び
【変更】よりフォーマル感を出しつつ、相談しやすさも

- ヘッダーに「専門家監修」的なバッジ
- 結果画面は「診断レポート」風のレイアウト
- セクションごとに明確な区切り線
- 最後に「無料相談」への誘導を強調
- 余白を広めに取り、落ち着いた印象に
```

### 2. 色使い: 信頼のネイビー + 相談のグリーン

```
【現状】ネイビー + グレーブルー
【変更】より洗練されたトーンに

- 背景: クールグレー (#F5F6F8)
- カード背景: 純白 (#FFFFFF)
- メインアクセント: ダークネイビー (#1E3A5F) - 信頼・専門性
- CTAアクセント: LINEグリーン (#06C755) - 相談への誘導
- テキスト: ほぼ黒 (#1A1A2E)
- サブテキスト: ミディアムグレー (#5C6878)
- 警告・注意: アンバー (#D97706)
- ADスコア高: ゴールド系 (#B8860B)
```

### 3. カードスタイル: フォーマル・クリーン

```
【現状】他ツールと同じ
【変更】より洗練されたビジネス感

- 角丸: 8px（控えめでフォーマル）
- シャドウ: 薄めでシャープ (0 2px 8px rgba(0, 0, 0, 0.06))
- ボーダー: 薄いグレー (1px solid #E2E5EA)
- カード内余白: 20px〜24px
- 見出しに下線アクセントを入れる
```

### 4. ADスコアメーター: 専門性の演出

```
【現状】単純なプログレスバー
【変更】より「診断結果」感を

- 半円型のゲージメーター（車のスピードメーター風）
- スコアに応じて色が変化（低=グレー、高=ゴールド）
- メーターの周りに目盛り
- 中央に大きくスコア数値
- 下に「交渉余地: 高」などのラベル
```

### 5. アイコン: ビジネス・不動産系

```
- Lucide Reactを継続使用
- 不動産系:
  - Building2, Home, Key, MapPin
  - FileText（契約書）, Calculator（計算）
- ビジネス系:
  - Briefcase, Shield（安心）, Award（実績）
  - Users（相談）, MessageCircle（LINE）
- 数値系:
  - TrendingDown（節約可能性）
  - Percent（手数料率）
  - BadgeJapaneseYen（金額）
```

### 6. 特徴的なUI要素

```
1. 診断レポートカード
   - 上部に「診断結果レポート」ヘッダー
   - 日付と診断IDを表示（信頼感）
   - 枠線で囲んだフォーマルなデザイン

2. ADスコアメーター（半円ゲージ）
   - 0-100のスコアを視覚化
   - ゴールド〜グレーのグラデーション
   - 「交渉余地」のラベル付き

3. 手数料比較表
   - 「上限（法定）」vs「適正目安」vs「交渉後」
   - 差額を強調表示
   - 「最大○○円お得に」のバッジ

4. LINE相談CTAボタン
   - 画面下部に固定表示（スティッキー）
   - LINEグリーンで目立たせる
   - 「無料で相談する」の文言
   - LINEアイコン付き

5. 信頼性バッジ
   - 「累計○○件の相談実績」
   - 「専門スタッフ対応」
   - 小さめだが目につく位置に
```

### 7. トーン&マナー

```
- 文言は丁寧語ベース（ですます調）
- 専門用語には簡単な説明を添える
- 「ご安心ください」「お気軽に」などの安心ワード
- 煽りすぎない（「今すぐ！」などは避ける）
```

## 実装時の注意

1. **共通コンポーネント（packages/shared）は変更しない**
   - このツール専用のコンポーネントをsrc/components/に作成

2. **tailwind.config.jsでカラーを上書き**
   ```js
   colors: {
     'bg-base': '#F5F6F8',
     'bg-card': '#FFFFFF',
     'text-main': '#1A1A2E',
     'text-sub': '#5C6878',
     'accent': '#1E3A5F',       // ダークネイビー
     'accent-light': '#EEF2F7',
     'line-green': '#06C755',   // LINE CTA用
     'border': '#E2E5EA',
     'success': '#059669',
     'warning': '#D97706',
     'gold': '#B8860B',         // ADスコア高
   }
   ```

3. **LINE誘導は最重要**
   - このツールの目的は「LINE相談への誘導」
   - CTAボタンは常に目立つ位置に

4. **既存機能は壊さない**
   - デザイン変更のみ、ロジックは触らない
